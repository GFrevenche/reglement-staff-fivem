# Règlement Staff — Serveur PvP FiveM

{% hint style="success" %}
Bienvenue dans le règlement officiel du staff. Ce document est obligatoire pour tous les membres de l'équipe.
{% endhint %}

## Pourquoi ce règlement ?

Ce document définit les **règles**, les **droits** et les **devoirs** de chaque membre du staff. Il garantit une modération juste, cohérente et professionnelle sur le serveur.

Tout membre intégrant l'équipe s'engage à respecter **l'intégralité** de ce règlement.

---

## Hiérarchie du Staff

| Rang | Niveau | Rôle principal |
| --- | --- | --- |
| Modérateur Test | 1 | Observation et apprentissage |
| Modérateur | 2 | Modération active du serveur |
| Administrateur | 3 | Gestion avancée et supervision |
| Fondateur | 4 | Direction et décision finale |
