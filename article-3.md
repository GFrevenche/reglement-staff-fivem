# Article 3 — Activité

{% hint style="info" %}
Un staff actif est un staff efficace. L'activité minimum garantit une bonne couverture du serveur.
{% endhint %}

---

**3.1** — Chaque membre du staff doit être **actif au minimum 3 jours par semaine** sur le serveur.

**3.2** — En cas d'absence prolongée (+ de 3 jours), le membre doit **prévenir un supérieur** à l'avance dans le channel dédié sur Discord.

**3.3** — Une absence non justifiée de **7 jours ou plus** entraîne une rétrogradation automatique.

**3.4** — Une absence non justifiée de **14 jours ou plus** entraîne un renvoi du staff.

**3.5** — Les absences justifiées acceptées sont : examens, vacances, problèmes personnels, problèmes techniques. Une **preuve peut être demandée** si nécessaire.

---

### Résumé des délais

| Durée d'absence | Conséquence |
| --- | --- |
| 3 jours sans prévenir | Rappel à l'ordre |
| 7 jours sans prévenir | Rétrogradation |
| 14 jours sans prévenir | Renvoi |

---

{% hint style="warning" %}
Prévenir = poster un message dans le channel #absences sur Discord AVANT l'absence.
{% endhint %}
