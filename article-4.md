# Article 4 — Utilisation des commandes

{% hint style="danger" %}
L'abus de commandes staff est l'une des fautes les plus graves. Toute utilisation doit être justifiée.
{% endhint %}

---

**4.1** — Les commandes staff doivent être utilisées **uniquement dans le cadre de la modération**. Toute utilisation personnelle est interdite.

**4.2** — Il est **interdit** d'utiliser des commandes staff (noclip, tp, vanish, give, spawn car, etc.) pendant une session de jeu PvP.

**4.3** — Tout **give d'item ou spawn de véhicule** doit être justifié (compensation, event, bug) et noté dans les logs.

**4.4** — Il est **strictement interdit** de se give des items, armes, véhicules ou argent à soi-même ou à ses amis.

**4.5** — Un membre du staff ne doit **jamais utiliser ses pouvoirs pour avantager** un joueur, un ami ou un groupe.

**4.6** — Chaque utilisation de commande est **automatiquement loguée**. Les logs sont vérifiés régulièrement par les supérieurs.

---

### Exemples d'abus

| Situation | Verdict |
| --- | --- |
| Se tp vers un joueur pour le surveiller | Autorisé |
| Se tp vers un ami pour jouer avec lui | Interdit |
| Give un item pour compenser un bug | Autorisé (avec preuve du bug) |
| Give une arme à un ami | Interdit |
| Spawn un véhicule pour un event staff | Autorisé |
| Spawn un véhicule pour jouer en PvP | Interdit |
| Noclip pour surveiller un suspect de cheat | Autorisé |
| Noclip pour traverser la map en jouant | Interdit |

---

{% hint style="warning" %}
En cas de doute, demandez l'autorisation à un supérieur AVANT d'utiliser une commande.
{% endhint %}
